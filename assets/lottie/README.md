# Lottie Animations

This directory is for Lottie animation JSON files.

## Where to Find Free Lottie Animations

- [LottieFiles](https://lottiefiles.com/) - The largest library of free Lottie animations
- [Icons8 Animated Icons](https://icons8.com/lottie-animations) - High-quality animated icons

## Suggested Animations for Welcome Screen

For the welcome screen, you can download and add animations related to:
- Card shuffling
- Sparkles/stars
- Coins
- Playing cards
- Game themes

## Usage in Code

To use a Lottie animation:

```dart
import 'package:lottie/lottie.dart';

Lottie.asset(
  'assets/lottie/your_animation.json',
  width: 200,
  height: 200,
  fit: BoxFit.contain,
)
```
